# KLineChartView

Android仿IQ Option K线图实现（包含MA,BOLL,MACD,KDJ,RSI,VOL指标）

本项目是在 https://github.com/fujianlian/KLineChart 基础上进行修改的，对UI展示进行了修改。

本项目模拟了socket实时推送数据
